# 创建课程阶段内容

按照以下步骤创建或更新课程阶段内容：

## 1. 确定阶段信息

确认要创建/更新的阶段目录，例如 `phase11-react-core`。

## 2. 创建目录结构

确保阶段目录包含以下结构：

```
src/phaseXX-name/
├── README.md      # 阶段概述（学习目标、核心知识点、学习资源）
├── CONCEPT.md     # 深度概念讲解（包含代码示例）
├── examples/      # 示例代码目录
└── exercises/     # 练习题目录
```

## 3. README.md 模板

```markdown
# Phase XX: [阶段名称]

## 📌 学习目标

- [ ] 目标 1
- [ ] 目标 2
- [ ] 目标 3

## 🔑 核心知识点

### 知识点 1

简要描述...

### 知识点 2

简要描述...

## 📚 学习资源

- [资源名称](链接)

## ⏱️ 预计学习时长

X 周（每周 10-15 小时）
```

## 4. CONCEPT.md 模板

```markdown
# [阶段主题] 深入解析

## 📌 一、[主题 1]

### 1. [子主题]

\`\`\`javascript
// 代码示例
\`\`\`

### 2. [子主题]

...

---

## 📌 二、[主题 2]

...

---

## 📚 推荐学习资源

| 资源 | 链接 |
| ---- | ---- |
| ...  | ...  |
```

## 5. 示例代码规范

- 使用现代 ES6+ 语法
- 添加详细中文注释
- 包含运行说明
- 提供渐进式示例（从简单到复杂）

## 6. 验证检查

// turbo
确认文件创建成功：

```bash
ls -la src/phaseXX-name/
```
