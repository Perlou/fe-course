## 🎯 核心概念

### Flux 架构

```
View → Action → Dispatcher → Store → View
 ↑                                    ↓
 └────────────────────────────────────┘
```

### 状态分类

```
Local State: 组件内部状态
Shared State: 跨组件共享状态
Server State: 服务端数据状态
URL State: 路由参数状态
```
