# 🎨 CSS 布局练习

## 项目概述

使用纯 CSS 实现一个响应式博客首页布局，练习 Flexbox、Grid、动画和媒体查询。

## 练习要求

### 页面布局

1. **导航栏** — Flexbox 水平布局，响应式汉堡菜单
2. **文章列表** — Grid 布局，卡片式设计
3. **侧边栏** — 标签云、分类列表
4. **页脚** — 多栏 Grid 布局

### 技术要求

- Flexbox 实现导航栏和页脚
- Grid 实现主内容区域 (文章卡片网格)
- 媒体查询实现响应式 (手机/平板/桌面)
- CSS 动画实现悬停效果
- CSS 变量实现主题色管理

### 响应式断点

```
手机:  < 768px  (单列)
平板:  768-1024px (双列)
桌面:  > 1024px (三列 + 侧边栏)
```
