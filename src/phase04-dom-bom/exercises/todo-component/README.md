# 🖱️ DOM & BOM 练习

## 项目概述

实现一个交互式 Todo 组件，练习 DOM 操作、事件处理、事件委托和本地存储。

## 练习要求

### 功能

1. **添加** — 输入框 + 回车/按钮添加待办
2. **完成** — 点击切换完成状态 (strikethrough)
3. **删除** — 点击删除按钮移除
4. **过滤** — 全部/未完成/已完成 筛选
5. **持久化** — localStorage 保存数据

### 技术要求

- 纯 DOM API (不使用框架)
- 事件委托处理列表点击
- `createElement` / `innerHTML` 动态渲染
- `localStorage` 数据持久化
- 自定义事件 (CustomEvent) 通信
