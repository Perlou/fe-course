# Phase 21: 性能优化

> **目标**：掌握前端性能优化  
> **预计时长**：2 周

## 📚 学习目标

1. 理解性能指标 (Core Web Vitals)
2. 掌握加载性能优化
3. 掌握运行时性能优化
4. 学会性能监控与分析

## 知识要点

- 性能指标 (LCP, FID, CLS, INP)
- 资源优化 (压缩、懒加载、预加载)
- 渲染优化 (避免重排重绘)
- 代码优化 (Tree Shaking、代码分割)
- 性能监控 (Performance API, Lighthouse)

## 核心概念

```
Core Web Vitals:
• LCP (Largest Contentful Paint) < 2.5s
• FID (First Input Delay) < 100ms
• CLS (Cumulative Layout Shift) < 0.1
• INP (Interaction to Next Paint) < 200ms
```

---

> 完成本阶段后，你应该能够分析和优化前端性能。
