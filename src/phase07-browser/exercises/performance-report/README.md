## 练习任务

1. 选择一个常用网站（如 GitHub, Bilibili, 知乎等）
2. 使用 Chrome DevTools 完成以上分析
3. 提出至少 3 条可行的优化建议
4. 如果可能，尝试用本地代理实施优化并对比效果

### 分析步骤

```
1. 打开 Chrome DevTools (F12)
2. Network 面板 → 禁用缓存 → 录制页面加载
3. Lighthouse 面板 → 运行审计
4. Performance 面板 → 录制用户交互
5. Memory 面板 → 拍摄堆快照
6. 记录数据到本报告模板
7. 分析瓶颈并提出建议
```
