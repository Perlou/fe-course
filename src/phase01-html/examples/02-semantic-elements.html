理解 HTML5 语义化标签

发布于 2024年1月15日

什么是语义化？

语义化 HTML 是指使用恰当的标签来描述内容的结构和含义， 而不仅仅是使用 div 和 span。

语义化的好处包括：

  • 提高代码可读性和可维护性
  • 有利于搜索引擎优化（SEO）
  • 提升无障碍访问体验
  • 便于团队协作

常用语义化标签

HTML5 引入了多个语义化标签：

对比示例

传统写法 vs 语义化写法：


<!-- 传统写法 -->
<div id="header">...</div>
<div id="nav">...</div>
<div id="main">...</div>

<!-- 语义化写法 -->
<header>...</header>
<nav>...</nav>
<main>...</main>
