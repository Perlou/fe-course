HTML 文档结构

1. DOCTYPE 声明

<!DOCTYPE html> 告诉浏览器这是一个 HTML5 文档。 这是文档的第一行，必须存在。

2. <html> 根元素

<html lang="zh-CN"> 是整个页面的根元素。 lang 属性用于指定页面语言，有助于 SEO 和无障碍访问。

3. <head> 头部

头部包含不可见的元信息：

  • charset - 字符编码（必须设置为 UTF-8）
  • viewport - 视口设置（响应式布局必需）
  • title - 页面标题（显示在标签页和搜索结果）
  • description - 页面描述（SEO 重要）
  • link - 外部资源链接（CSS、图标等）
  • style - 内部样式
  • script - JavaScript 脚本

4. <body> 主体

主体包含所有用户可见的内容：文本、图片、链接、表单等。
