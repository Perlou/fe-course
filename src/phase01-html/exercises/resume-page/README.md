# 🌐 HTML 基础练习

## 项目概述

使用纯 HTML 搭建一个个人简历页面，练习语义化标签、表单、多媒体等核心知识。

## 练习要求

### 页面内容

1. **个人信息** — 姓名、头像、联系方式
2. **教育经历** — 使用表格展示
3. **技能列表** — 使用有序/无序列表
4. **项目经历** — 使用语义化标签 (article, section)
5. **联系表单** — 包含各种表单元素

### 技术要求

- 使用 `<header>`, `<main>`, `<footer>`, `<nav>` 语义化结构
- 使用 `<table>` 展示教育经历
- 使用 `<form>` 实现联系表单 (input, select, textarea)
- 使用 `<img>`, `<picture>` 展示图片
- 使用 `<a>` 实现页内锚点导航

- [ ] HTML5 语义化标签
- [ ] 表单元素与验证
- [ ] 表格结构
- [ ] 多媒体嵌入
