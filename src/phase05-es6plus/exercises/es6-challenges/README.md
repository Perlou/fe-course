# ✨ ES6+ 练习

## 项目概述

使用 ES6+ 特性重构和实现一组实用工具函数，练习解构、Promise、迭代器、Proxy 等。

## 练习列表

### 1. Promise 练习

- 实现 `Promise.all()`
- 实现 `Promise.race()`
- 实现 `Promise.allSettled()`
- 实现重试函数 `retry(fn, times)`

### 2. 迭代器与生成器

- 实现 Range 迭代器 `range(start, end, step)`
- 实现无限序列生成器 `fibonacci()`
- 实现异步迭代器 `asyncMap(iterable, fn)`

### 3. Proxy 练习

- 实现响应式对象 `reactive(obj)`
- 实现数据验证 Proxy
- 实现负数索引数组 `arr[-1]`

### 4. 模块化练习

- 实现 EventEmitter (模块导出)
- 实现插件系统 (动态 import)

- [ ] 解构赋值
- [ ] Promise / async-await
- [ ] 迭代器协议
- [ ] Proxy / Reflect
