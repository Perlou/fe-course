# 📊 数据可视化仪表盘实战

## 项目概述

使用 Canvas 和 SVG 构建一个交互式数据可视化仪表盘，展示多种图表和动画效果。

## 技术要求

```
Canvas: 柱状图、折线图、粒子动画
SVG: 饼图、仪表盘、进度环
交互: 鼠标悬停高亮、点击钻取、tooltipX
响应式: 自适应容器大小
```

## 功能需求

### 图表组件

1. **柱状图 (Canvas)** — 多系列、动画入场
2. **折线图 (Canvas)** — 数据点 + 平滑曲线
3. **饼图 (SVG)** — 扇形 + 文字标签
4. **仪表盘 (SVG)** — 圆弧进度指示
5. **热力图 (Canvas)** — 颜色矩阵

### 交互功能

1. 鼠标悬停显示 tooltip
2. 点击扇形展开明细
3. 数据切换动画 (缓动)
4. 容器 resize 自适应

## 实现步骤

### Step 1: 基础绘图引擎

1. Canvas 2D 工具函数
2. SVG 构建器
3. 比例尺与坐标转换

### Step 2: 图表组件

1. BarChart (Canvas)
2. LineChart (Canvas)
3. PieChart (SVG)
4. GaugeChart (SVG)

### Step 3: 交互与动画

1. 碰撞检测 (鼠标位置)
2. 缓动动画 (easing)
3. Tooltip 层
4. 响应式 resize
