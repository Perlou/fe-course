# Phase 20: 微前端

> **目标**：掌握微前端架构  
> **预计时长**：2 周

## 📚 学习目标

1. 理解微前端概念与适用场景
2. 掌握主流微前端方案
3. 实现应用间通信
4. 处理样式隔离与 JS 沙箱

## 知识要点

- 微前端架构设计
- qiankun / Module Federation
- 样式隔离方案
- JS 沙箱原理
- 应用间通信

## 核心概念

```
微前端: 将单体应用拆分为多个可独立开发、部署的子应用

主应用 (Base App)
├── 子应用 A (React)
├── 子应用 B (Vue)
└── 子应用 C (Angular)
```

---

> 完成本阶段后，你应该能够设计和实现微前端架构。
