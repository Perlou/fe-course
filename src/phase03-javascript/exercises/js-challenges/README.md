# 🧮 JavaScript 基础练习

## 项目概述

实现一组 JavaScript 核心知识的编程练习，巩固作用域、闭包、原型链、this 等概念。

## 练习列表

### 1. 闭包练习

- 实现 `createCounter()` — 计数器工厂
- 实现 `once(fn)` — 只执行一次的函数
- 实现 `memoize(fn)` — 缓存函数结果

### 2. 原型链练习

- 实现 `new` 操作符
- 实现 `instanceof` 操作符
- 手写类继承 (ES5 组合寄生)

### 3. this 练习

- 实现 `Function.prototype.bind`
- 实现 `Function.prototype.call`
- 实现 `Function.prototype.apply`

### 4. 综合练习

- 实现 `deepClone(obj)` — 深拷贝
- 实现 `flattenArray(arr)` — 数组扁平化
- 实现 `curry(fn)` — 函数柯里化

## 学习要点

- [ ] 闭包与作用域链
- [ ] 原型链与继承
- [ ] this 绑定规则
- [ ] 高阶函数
